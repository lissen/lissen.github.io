How to use

Download the source files and build it with Visual Studio, then run the LissenRepl.exe console program to start playing with Lissen.

Here are some commands you can try:

> (+ 1 2)
3
> (define (add x y) (+ x y))
Lissen.LambdaExpr
> (add 3 5)
8
